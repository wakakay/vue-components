#### 目录介绍
```
  |--build   构建脚本
  |--config  构建配置
  |--node_modules   // 依赖node工具包
  |--static         // 静态模板
  |--src            //  源码
      |--assets        // 资源
          |--images       // 公共图片
          |--styles       // 公共样式
              |--iconfont // 公共字体图标    
      |--componets     // 公共组件
      |--pages         // 页面组件 
          |--user
      |--router        // 配置页面路由
      |--store         // vuex状态
      |--App.vue       // 页面入口文件
      |--main.js       // 程序入口文件，加载各种公共组件
```
